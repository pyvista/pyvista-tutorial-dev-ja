PyVistaとQT#

PyVista を使用して， pyinstaller と Qt フレームワークを使用したスタンドアロンアプリケーションを作成する方法を説明します．

Tip

チュートリアルのこのセクションは，PyVistaドキュメントの pyvistaqt documentation と Extras の章から採用されています．

PyVistaとQTの併用#

PyVistaは，QTにプロットを置くためのインタフェース (QT参照) を pyvistaqt ライブラリに持ち，QTとVTKを組み合わせることができるようにしました． これにより，Qt を使用する際のメッシュの追加，更新，制御が簡素化されます．

pip を使ったインストールは:

pip install pyvistaqt

このパッケージを conda と共にインストールするには次のように実行します．

conda install -c conda-forge pyvistaqt

インストールしたら，他のPyVistaプロッタと同じように pyvistaqt.BackgroundPlotter を使ってください．

簡単な例#

pyvistaqt.BackgroundPlotter のインスタンスを作成し，球体を描画します．

import pyvista as pv
from pyvistaqt import BackgroundPlotter

sphere = pv.Sphere()

plotter = BackgroundPlotter()
plotter.add_mesh(sphere)

背景のプロット#

通常の PyVista のプロットウィンドウはブロッキング動作をしますが， pyvistaqt.BackgroundPlotter を使ってバックグラウンドでプロットし，リアルタイムでプロッタを更新することが可能です． これは pyvistaqt を必要としますが，それ以外は通常の PyVista pyvista.Plotter インスタンスのように表示され，機能します．例えば:

import pyvista as pv
from pyvistaqt import BackgroundPlotter

sphere = pv.Sphere()

plotter = BackgroundPlotter()
plotter.add_mesh(sphere)

# can now operate on the sphere and have it updated in the background
sphere.points *= 0.5

マルチプロッタ#

次の例は，複数のプロッターを持つインターフェースを使用する方法を示しています．それぞれのプロッタは選択することができ，通常のPyVista pyvista.Plotter のインスタンスのように機能します．

import pyvista as pv
from pyvistaqt import MultiPlotter

mp = MultiPlotter(nrows=2, ncols=2)
mp[0, 0].add_mesh(pv.Sphere())
mp[0, 1].add_mesh(pv.Cylinder())
mp[1, 0].add_mesh(pv.Cube())
mp[1, 1].add_mesh(pv.Cone())

pyinstallerでPyVistaをフリーズさせる#

pyinstaller と PyVista を使って素晴らしいスタンドアロンプログラムを作ることができますし， PyQt5 や pyside2 を組み込んでグラフィカルユーザーインターフェースを作ることさえ可能です． VTKのバージョンによっては，セットアップにいくつかの追加手順が必要です．

VTK v9 を実行する場合，いくつかの hiddenimports を追加する必要があります． 分かりやすくするために， Using Spec Files の説明に従って，specファイル(ここでは pyvista.spec という名前にします)を作成します． Analysis を修正し，以下の hidden imports を追加してください:

main_py = os.path.join(some_path, 'main.py')
a = Analysis([main_py],
             pathex=[],
             binaries=[],
             hiddenimports=['vtkmodules',
                            'vtkmodules.all',
                            'vtkmodules.qt.QVTKRenderWindowInteractor',
                            'vtkmodules.util',
                            'vtkmodules.util.numpy_support',
                            'vtkmodules.numpy_interface.dataset_adapter',
                           ],

そこからPyVistaを使ってアプリケーションをフリーズさせ，スタンドアロンアプリケーションを作ることができます．

事例紹介#

VesselVio は，セグメント化された血管系データの解析と可視化のためのオープンソースアプリケーションです．

MNE ヒトの神経生理学的データを探索，視覚化，分析するためのオープンソースの Python パッケージです．MEG, EEG, sEEG, ECoG, NIRS, など．

femorph は，特に光学スキャンデータに合わせて有限要素モデルを更新するのに適したメッシュ変成ソフトウェアです．
